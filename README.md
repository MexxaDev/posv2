# Levitar POS - GG Beach House

Sistema de Punto de Venta

## Cómo ejecutar

### Windows
Doble click en `levitar-pos.bat`

### macOS
Doble click en `levitar-pos.command`

## Importante

- La primera vez que abras la app, los datos se cargan desde los archivos JSON en la carpeta `data/`
- Los datos se guardan en el navegador (IndexedDB)
- Haz backup regularmente con el botón **Exportar** en la sidebar

## Instalar como app

Chrome/Edge:
1. Abre la app en el navegador
2. Presiona `F11` para pantalla completa
3. O haz click derecho → "Instalar Levitar POS"

## Backup

**Exportar**: Click en botones Exportar → descarga archivo JSON con todos los datos
**Importar**: Click en Importar → selecciona archivo JSON de backup

## Requisitos

- Navegador: Chrome, Edge, Firefox, Safari (última versión)
- NO requiere servidor
- NO requiere internet
